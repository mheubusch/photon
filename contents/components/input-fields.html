Input Fields

Input fields allow users to type in text information.

Usage

Use input fields when requiring users to enter a text information like email address, account name, searching keyword… that couldn’t be provided for users to select.

Types

Text Field

A text field can validate the entered data and provide hints of how to meet the certain criteria.

Search Field

A search field can either live-search results or present search results after users send a query.
