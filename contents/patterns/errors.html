Errors

It's important to distinguished three different kind of error:

  • User input errors
  • System errors

Components using errors states:

  • Message bars
  • Notification bars
  • Input field

Usage

Errors provide short and relevant information about a specific situation. If possible, give to the user an action that will help them address the warning.

Placement

Errors are usually place close to the context.

Style

Icon

The icon used for indicate errors is an exclamation mark enclosed within a triangle, usually white on a red background, or red.

Colors

Red is the dedicated color for errors states. Warning text should be legible, with noticeable contrast against its background color.

Background

Red 50 #ff0039, or trasparent

Content

White #ffffff, or Red 60 #d70022
